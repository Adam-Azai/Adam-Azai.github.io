Green de la Creme

Overview

Green de la Creme is a group project whose goal was to bring together plant lovers to share their beloved plants, ask questions about plant care and foster a community amongst themselves. The group consisted of myself, Adam Azai and my three other Hack Reactor colleagues, Shayne Blanc, Regan Ternasky, and Brandon Gomez. We developed Green de la Creme following agile principles, holding daily standups, writing concise user stories, issuing tickets and creating easy to test merge requests.

The main features of the site are the blog, forum, plant database and plant resources page. I was responsible for the forum, plant database and half of the plant resources page for this project. The forum is composed of threads that are created by only authorized users of the site, a user who has created an account and is logged in. The threads are designed to be questions or tutorials for Plant care that can be searched throughout the forum. A thread is composed of a Title, Body and Image where the user can address the topic of the thread and give more details in the body. Viewing the forum at first only shows the list of threads with their titles's only, upon selecting a thread the details of the thread as well as any replies to the thread can be viewed in a new page.

Plant Data

The Plant Data page contains a search feature for users to learn more details regarding any plant. The search function utilizes a third-party API known as the Perenual API. The Perenual API offers an abundant amount of botanical information for developers. I set up a search bar where users would input the name of a plant they would like to learn more about. The search bar takes that input and places it into the q parameter of the Perenual Plant list Api query, and returns a list of plants whose common name contains the input value. The list of plants that match the name are formatted into a grid of cards that can be clicked on to view more informations, such as the watering frequency, growth cycle, and the amount of sunlight the plant needs to grow.

Plant Resources

The Plant Resources Page is a simple page with links to youtube videos and articles about plant care. The hero of the page is a Swiper Carousel autoplaying beautiful svg images provided by Pexels. The lower half of the Resources Page contains a Yelp fusion API feature that was implemented by my teammate, Shayne Blanc. If a user had inputted location data upon their account creation, then local nurses/gardens around them will be shown by a carousel of cards displaying various information regarding the plant businesses.
